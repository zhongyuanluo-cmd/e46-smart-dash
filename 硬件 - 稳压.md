
## CAN/K-Bus稳压

** 需要车载级防护，不是简单的稳压。**

```
E46 电气环境实测：

  正常运转:   13.8V - 14.4V
  冷启动瞬态: 最低 6V（持续 50-100ms）
  抛负载瞬态: 最高 60-80V（持续 200-400ms）
  交流纹波:   发电机 50-200mV 纹波叠加
```

### 推荐方案：单路宽压模块供电所有总线外设

```
KL30常电(9-16V)
    │
    ▼
┌──────────────────────┐
│ 车载宽压 buck         │
│ MP4580 / LMR14030    │
│ 输入: 4.5-40V        │
│ 输出: 5V / 3.3V      │
│ + TVS管 (SMBJ30A)    │
│ + 防反接 (理想二极管)  │
└──────────┬───────────┘
           │ 5V
    ┌──────┼──────┬──────────┐
    ▼      ▼      ▼          ▼
  MCP2515-E  TH3122.4 TH3122.4 其他5V外设
  (CAN)     (K-Bus)  (I-Bus)
```

| 防护措施 | 作用 | 成本 |
|----------|------|:---:|
| TVS SMBJ30A | 吸收抛负载尖峰 | ¥0.5 |
| 理想二极管 LM74700 | 防反接，零压降 | ¥2 |
| π型滤波 (LC) | 滤除发电机纹波 | ¥0.3 |
| 共模扼流圈 (CAN 线) | CAN 总线专用抗扰 | ¥1 |

> **CAN 收发器和总线接口必须用独立于音频系统的电源轨**，防止总线噪声耦合到音频。


## 音频稳压

**需要，而且要比总线更讲究。**

```
音频系统的敌人：

  发电机 whine   → 随转速变化的 1kHz-5kHz 啸叫（最烦人）
  数字地弹      → I2S/MIPI 的开关噪声耦合到模拟
  CAN 脉冲      → 总线收发时的电流尖峰
```

### 推荐方案：独立 LDO + 分离地

```
系统 5V
    │
    ▼
┌─────────────────┐
│ 磁珠 BLM18PK     │  ← 高频隔离（>1MHz 噪声衰减）
│ (600Ω@100MHz)   │
└────────┬────────┘
         ▼
┌─────────────────┐
│ LDO LP5907       │  ← 超低噪声 LDO（6.5μV RMS）
│ 5V → 3.3V       │     高 PSRR（82dB@1kHz）
│                  │     专门给 DAC + 运放
└────────┬────────┘
         ▼
    PCM5102A DAC (AVDD)
    模拟地平面（与数字地单点接地，磁珠桥接）
```

| 措施 | 效果 | 成本 |
|------|------|:---:|
| 独立超低噪 LDO | 纹波 < 10μV | ¥1.5 |
| 磁珠隔离 | 阻断 >1MHz 数字噪声 | ¥0.2 |
| 模拟/数字地分离 | 消除地回路哼声 | 0（PCB设计） |
| 音频走线包地 | 抗空间辐射耦合 | 0（PCB设计） |
