### 进程架构

```
systemd 管理以下核心守护进程：

can-gateway.service      ─── 常驻 ─── CAN 消息接收、解码、转发
sensor-collector.service ─── 常驻 ─── GPS + IMU + TPMS 统一采集
voice-engine.service     ─── 常驻 ─── 唤醒词监听 + ASR + NLU + TTS
audio-manager.service    ─── 常驻 ─── PulseAudio + BlueZ + 混音
track-logger.service     ─── 赛道时启动 ─── 高频数据记录
carplay.service          ─── 可选常驻 ─── CarPlay 协议栈
ui-app                   ─── 常驻 ─── Qt6/QML 主界面

所有进程通过 DBus 或 Unix Socket 通信。
```
