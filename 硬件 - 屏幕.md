## 屏幕类型选择

E46 屏幕装在烟灰缸位置，驾驶员视线从上往下约 **30-45°** 斜视。TN 屏从这个角度看会严重偏色发白，**只有 IPS 能满足车载倾斜视角的可读性要求**。

| 类型 | 可视角度 | 阳光下可读 | 耐温 | 适合车载？ |
|------|:---:|:---:|:---:|:---:|
| **IPS** | 178° 全方向 | 需高亮背光 | 70-85°C | ✅ **唯一选择** |
| TN | 窄角，斜看发白 | 差 | 一般 | ❌ 视角不满足 |
| VA | 中等 | 中等 | 一般 | 🟡 视角不如 IPS |
| OLED | 178° | 强光下亮度不足 | ❌ 怕高温烧屏 | ❌ 车内 70°C 加速老化 |
| QLED | 178° | 好 | 类似 IPS | 🟡 好但小尺寸极少，贵 |

**结论**：ER-TFT069-1 的 IPS 面板是正确的选择。

### 几个需要关注的要点

| # | 要点 | 分析 |
|---|------|------|
| 1 | **MIPI 4-lane** | 屏是 4-lane，Core3566 两个 MIPI 接口各 2-lane。但 280×1424@60fps 带宽用 **2-lane 就够了**（约 574Mbps，2-lane 可跑 ~1Gbps），需要确认 JD9365DA 是否支持 2-lane 模式 |
| 2 | **背光 12V** | 系统是 5V，需要 boost 升压到 12V 或直接从车 12V 取电（但需滤波） |
| 3 | **逻辑 2.8V** | 需要一个 3.3V→2.8V LDO |
| 4 | **工作温度上限 70°C** | 夏天车内可达 60-70°C，刚好踩线，需注意散热 |
| 5 | **触摸是 I2C** | 直接接 Core3566 I2C，驱动成熟 |


### 带宽计算

```
280 × 1424 × 60fps × 24bpp = 574 Mbps（有效像素）

加上消隐期（~20%）：
574 × 1.2 ≈ 689 Mbps 实际需求

MIPI DSI 2-lane 能力：
  保守: 400 Mbps/lane × 2 = 800 Mbps  ✅ 够
  典型: 500 Mbps/lane × 2 = 1000 Mbps ✅ 充裕
```

### JD9365DA 控制器的 lane 配置

JD9365DA 是奕力科技的 TFT 驱动 IC，支持通过寄存器配置 lane 数：

| 寄存器 | 功能 |
|--------|------|
| `DSI_LANE` | 0x02 = 2-lane, 0x04 = 4-lane |

### 关键电路设计

#### 1. MIPI DSI 走线

```
关键规则：
├── 差分阻抗: 100Ω ±10%
├── 等长匹配: D0/D1/CLK 对内长度差 <0.5mm
├── D0/D1/CLK 对间长度差 <5mm
├── 参考地平面连续，不跨分割
└── 差分对间距 >2×线宽

2-lane 模式: D2/D3 在 FFC 端悬空即可
```

### 电源

```
+5V (引脚77,79)  ──►  系统 5V
   │
   ├──► LDO → 3.3V → 触摸 VDD (FFC Pin 6)
   │
   ├──► LDO → 2.8V → 屏 VDD (FFC Pin 22)
   │
   └──► MT3608 boost → 12V → 背光 LEDA (FFC Pin 3-4)
                             背光 LEDK → GND (FFC Pin 6-7)
```

#### 2. 背光驱动

更简单的方案：直接用 **MT3608 升压模块**（¥0.5）

```
5V ──► MT3608 ──► 12V ──► 限流电阻 10Ω ──► LEDA
                │
          PWM ← GPIO（控制 EN 脚调亮度）
```

#### 3. 逻辑电源 2.8V

```
3.3V ──► LDO (XC6206P282MR) ──► 2.8V → VDD (Pin 22)
                                   │
                                  1μF + 10μF 退耦
                                  
触摸 VDD 直接用 3.3V → 触摸 Pin 6
```


### IO 资源占用总结

| 接口 | Core3566 引脚 | 用途 |
|------|:---:|------|
| MIPI DSI1 | 175-196 (10 pins) | 显示 |
| I2C3 | 35, 36 | 触摸 |
| GPIO | 24, 48, 49, 50 | 触摸INT/RST, 屏RESET/STBYB |
| GPIO (PWM) | 任意 | 背光调光 |

## 安装位置验证计划

> 当前方案：屏幕装在**烟灰缸位**（最底层，倾斜安装）。
> 备选方案：屏幕装在**眼镜盒位**（上升一格，240×60mm 更宽裕但垂直安装）。
