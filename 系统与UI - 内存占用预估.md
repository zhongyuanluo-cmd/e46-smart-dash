
### 内存占用预估（Linux + 你的全部功能）

| 组件 | 内存占用 | 说明 |
|------|:---:|------|
| Linux 内核 + systemd | ~150MB | 精简内核 |
| GPU/显示缓冲 | ~20MB | 280×1424×4 双缓冲 |
| PulseAudio + BlueZ | ~40MB | 音频+蓝牙栈 |
| LVGL / Qt 运行时 | ~80MB | UI 框架 |
| UI 应用 + 资源 | ~100MB | 界面图片/字体 |
| ASR 声学模型 | ~15MB | NPU 运行时 DDR 加载 |
| NLU 模型 | ~10MB | 语义理解 |
| TTS 模型 (Piper) | ~30MB | 离线语音合成 |
| 传感器采集线程 | ~10MB | GPS+IMU+CAN+TPMS |
| CAN 消息缓冲/日志 | ~50MB | 环形缓冲 |
| SQLite / 数据存储 | ~30MB | 赛道记录 |

| **实际使用** | **~735MB** | |
