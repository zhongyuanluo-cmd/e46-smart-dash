### 推荐：Armbian 开发 → Buildroot 量产

```
阶段一：原型开发 (1-3个月)
  Armbian (Debian 12)
  ├── apt install 一切 → 快速试错
  ├── 微雪官方支持，镜像直接烧录
  └── 开发完成后，知道需要哪些组件

阶段二：产品化 (第4个月起)
  Buildroot
  ├── 只编译需要的包 → 镜像 ~150MB
  ├── 启动时间压缩到 5s
  └── 裁掉 apt/systemd-udev/NetworkManager 等非必要组件
```

| | 开发阶段 | 产品化阶段 |
|------|:---:|:---:|
| OS | Armbian | Buildroot |
| 内核 | 5.10/6.1 (Rockchip BSP) | 同，锁定版本 |
| NPU 驱动 | RKNN 1.6+ | 同 |
| GPU | Mali-G52 (Panfrost/Lima) | 同 |
