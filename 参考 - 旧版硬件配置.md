
# 2.1 主控S3
## 2.1.1 主控板（ESP32 S3 N16R8）板载设备：
 - 4GHz Wi-Fi (802.11 b/g/n) 
 - Bluetooth 5 (LE)
 - 双麦克风（用不到）
 - QMI8658 （六轴惯性测量单元）
 - PCF85063 RTC （时钟芯片，支持时间保持功能）
 - SD MMC 槽
 - IPEX 1 代座子 （可通过拆焊电阻切换为使用外部天线）
 - TCA9554PWR （8 位 I2C GPIO 扩展芯片）
 - W25Q128JVSI （16MB Flash 存储器）
 - ES8311 DAC （音频）
 - ES7210 ADC （音频）
 - 显示：分辨率：172 × 640     驱动芯片：AXS15231B    显示接口：QSPI    触摸接口：I2C
## 2.1.2 K/I-Bus交互：
MCP2003、MC33660 或 TH3122 + 24V 单向 TVS 二极管
## 2.1.3 PT-CAN总线交互：
SN65HVD230 或 TJA1050 + PESD1CAN（或者常用的双路汽车级 TVS 二极管对）
 ESP32-P4 + TJA1050 + isotp-c  [GitHub - lishen2/isotp-c: An implementation of the ISO-TP (ISO15765-2) CAN protocol in C · GitHub](https://github.com/lishen2/isotp-c)
## 2.1.4 系统级音频播放：
MAX98357A + 外接独立小喇叭
## 2.1.5 降压稳压：
1A 自恢复保险丝 + LM2596HV 宽压降压模块 + SMBJ15/6A  TVS 二极管（浪涌保护）
## 2.1.6 GPS：
ATGM336H
## 2.1.7 胎压：
TI CC1101
## 2.1.8 电源：
1SMA5934 级别的稳压/TVS 二极管 + DC-DC 降压芯片
## 2.1.9 语音拾取
Codec (ES8388) （外挂音频 Codec 做硬件波束成形，一颗音频 Codec 同时服务 P4 (ASR) 和 BT1026C (通话)）+阵列双麦
