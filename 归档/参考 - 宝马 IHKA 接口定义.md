宝马 E46 的 IHKA 后面确实有 4 个插头。核心信号（I-Bus、12V 常火、12V ACC 钥匙火、GND），在其中的X608和X610两个接头上。

[BMW WDS Online](https://bmwteka.com/wds/en/e46/4b2141af)

### IHKA 接口接线定义表

| **信号类型**               | **插头编号**           | **引脚号 (Pin)**           | **官方信号定义与常用线色参考**                                |
| ---------------------- | ------------------ | ----------------------- | ------------------------------------------------ |
| **GND (搭铁地线)**         | **X610** (18针大插头)  | **Pin 13** 或 **Pin 14** | Ground 基准接地点（通常为 **纯棕色 BR**）                     |
| **常电 (Terminal 30)**   | **X610** (18针大插头)  | **Pin 9**               | 压缩机继电器侧供电端（通电池常电，通常为红底条纹线）                       |
| **钥匙电 (Terminal 15)**  | **X608** (5/6针小插头) | **Pin 2**               | 连至车身 28 号保险丝（仪表点火电，通常为 **绿/蓝 GN/BL** 或绿底条纹）      |
| **I-Bus / K-Bus (总线)** | **X610** (18针大插头)  | **Pin 8**               | I/K-bus signal 车身总线信号（通常为 **白/红/黄三色线 WS/RT/GE**） |

**利用钥匙火（**X608** Pin 2）做全自动开关机**： 汽车停放时，12V 常火（Pin 2）会一直耗电。建议把 **Pin 2（钥匙火）** 接到你的 DC-DC 降压模块的 EN（Enable 启动引脚）或者直接作为降压输入。这样只要钥匙拧到 1 档或 2 档，S3 主控板才通电启动；拔掉钥匙，主控板彻底断电，完全不伤原车电瓶。

**MCP2003 的电压匹配**： 记得把 MCP2003 的 VBB 引脚接在 12V 电源上，而它的 VCC 必须接 ESP32-S3 的 **3.3V**（切勿接 5V），这样它的 RXD/TXD 吐出的就是安全的 3.3V TTL 电平，可以直接安全地插在 S3 的 GPIO 上。


**关于 I-Bus/K-Bus 物理层**：在 E46 的车身网络里，I-Bus 和 K-Bus 在物理层和协议层完全是连通的（由仪表盘 K-G-I 网关桥接，或者直接并线）。你把 ESP32 + MCP2003 的收发端挂在 **X610 的 Pin 8** 上，就能完美抓到全车（包括多功能方向盘、仪表盘、车身模块）的 K/I-Bus 报文。
    
**用万用表测一下再破线**：德国车的线束颜色有时会因为生产年份或车型（比如前期款 M52 相比后期款 M54，或者美规改版）出现细微变动。在正式接线给 ESP32 供电前，建议用万用表戳进端子孔复测一次：
- 拔下钥匙时：Pin 13/14 对车身金属件导通（电阻接近 0 欧姆）；Pin 9 能量出稳定的 12V。
- 拧开钥匙到 2 档（点火前一档，仪表盘全亮）时：Pin 2 才会跳出 12V；关掉钥匙则断电。
