**需要做的散热强化措施**：

1. 屏幕背面强制主动散热：

```
┌─────────────────────────────────────────┐
│          🖥️ 屏幕前面板 (玻璃)           │
├─────────────────────────────────────────┤
│          LCD 面板                        │
├─────────────────────────────────────────┤
│          驱动 IC (JD9365DA) ← 最怕热     │
├─────────────────────────────────────────┤
│   ╔══════ 铜均热板 ══════╗             │
│   ║  覆盖整个屏幕背面     ║             │
│   ╚══════════════════════╝             │
│          │          │                    │
│    ┌─────┴────┐ ┌─────┴───────┐        │
│    │  散热片   │ │ 3010 风扇    │        │
│    │  (被动)   │ │ (主动侧吹)   │        │
│    │          │ │ Sunon        │        │
│    │          │ │ MF30100V1    │        │
│    └──────────┘ └─────────────┘        │
└─────────────────────────────────────────┘
                    ↓ 热空气
            中控内部空腔 → 自然上升 → 顶部缝隙排出
```

2. 变速箱热隔离——气凝胶隔热垫（3mm，导热系数 ~0.02 W/m·K）：

```
剖面图（从车头方向看）：
  中控面板
    │
    ├── 🖥️ 屏幕
    ├── 屏幕背板散热组件
    ├── ═══════════════════  ← 3mm 气凝胶隔热垫
    ├── 中控塑料底板
    │
══════════ 变速箱通道隆起 ══════════
            ↑ 热源 50-65°C（行车时）
```

3. Core3566 与屏幕物理分离（不要堆叠）：

```
❌ 不要这样:
  ┌──────────┐
  │  屏幕     │
  ├──────────┤
  │ Core3566 │  ← SoC 热量直接传导给屏幕
  └──────────┘

✅ 应该这样:
  ┌──────────┐
  │  屏幕     │  ← 独立散热通道
  ├──────────┤
  │ 均热板    │
  ├──────────┤
  │ 隔热层    │  ← 阻断热传导
  ├──────────┤
  │ ~20mm 空隙│  ← 空气隔热 + 走线
  ├──────────┤
  │ Core3566  │  ← 独立散热通道
  └──────────┘
```

4. 屏幕遮阳罩（3D 打印，同时改善可读性）：

```
      前挡风玻璃
         ╱
    ☀️ 阳光
         ╲
   ┌─────┐
   │遮阳罩│  ← 中控面板上方延伸 15-20mm
   └─────┘
  ╔═══════╗
  ║ 屏幕  ║
  ╚═══════╝
```

6. 两个 DS18B20（屏幕背板 + SoC 散热片）
    
7. 熄火后散热逻辑：
    

```
车辆熄火 (ACC OFF)
    │
    ▼
T_screen > 50°C 且 T_screen_rising?
    │
    ▼
T_screen > 70°C 或 T_screen_rising > 0.5°C/min ?
    ├─ YES → 风扇全速 30 秒 → Suspend-to-idle
    │        (可选超级电容 1F/5.5V ¥3 独立供电)
    └─ NO  → 正常 Suspend-to-idle
```
