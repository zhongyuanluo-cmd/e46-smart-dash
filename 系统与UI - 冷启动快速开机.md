
低功耗待机方案

### E46 电源时序

```
ACC/KL15 状态：

  钥匙 OFF (位置0):   KL15=0V, KL30=常12V   系统断电
  钥匙 ACC (位置1):   KL15=12V              音响/点烟器有电
  钥匙 ON  (位置2):   KL15=12V              全车通电
  启动     (位置3):   KL50=12V (瞬态)        启动马达
  
  E46 休眠: ACC OFF 后 ~16分钟后整车总线休眠
```

### 推荐方案：三级电源管理

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│  KL15 ON (开车/ACC)                                  │
│  ├── RK3566 全速运行                                 │
│  ├── 所有外设正常供电                                 │
│  └── 功耗: ~5W (CarPlay) / ~3W (纯UI)               │
│                                                     │
│  KL15 OFF, < 16分钟 (短暂停车)                        │
│  ├── RK3566 suspend-to-idle (冻结用户空间)            │
│  ├── CAN/TPMS 保持监控 (唤醒源)                       │
│  ├── 功耗: ~0.3W (50mA@5V)                          │
│  └── 恢复时间: < 1秒                                 │
│                                                     │
│  KL15 OFF, > 16分钟 (过夜/长期)                       │
│  ├── RK3566 完全关机                                  │
│  ├── 外挂低功耗 MCU 监控 CAN 唤醒                      │
│  ├── 功耗: < 0.01W                                  │
│  └── 冷启动: 8-15秒（可优化到 5-8秒）                 │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 冷启动加速技巧

| 优化项 | 效果 | 说明 |
|--------|:---:|------|
| **裁减内核** | -2s | 只保留需要的驱动，去掉 USB/PCIe 等 |
| **Initramfs** | -2s | 根文件系统预加载到 RAM |
| **取消不必要的 systemd 服务** | -3s | 不要 network-manager、蓝牙配对等 |
| **U-Boot 快速引导** | -1s | 跳过设备树探测，硬编码配置 |
| **eMMC HS400 模式** | -1s | 最大化 eMMC 读速度 |
| **总目标** | **5-8秒** | 从 KL15 ON 到 UI 显示 |



### 补充：BlueZ 自动重连

```
熄火前连着手机A → BlueZ 记住配对信息
下次开机 → BlueZ 自动重连手机A（无需用户操作）
用户想换手机B → 进设置 → 启动配对Agent → 配对B

99% 的时间不需要配对Agent，开机时不需要它。
```

> **总结**：蓝牙配对 100% 保留，只是把配对 Agent 从开机链移到按需启动。抱歉之前表述不清！
