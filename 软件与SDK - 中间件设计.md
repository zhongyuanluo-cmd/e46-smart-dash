#### 4.1 CAN 网关

```
┌─────────────────────────────────────────────────┐
│              can-gateway.service                 │
│                                                  │
│  SocketCAN (can0: PT-CAN, 500kbps)              │
│  │                                               │
│  ├── CAN 帧接收线程 (实时)                        │
│  │   ├── 0x316: 发动机转速 (RPM)                 │
│  │   ├── 0x329: 车速                             │
│  │   ├── 0x1F0: 水温                             │
│  │   ├── 0x1F5: 油温                             │
│  │   ├── 0x545: 空燃比 AFM                       │
│  │   ├── 0x1F3: 档位                             │
│  │   └── ... (BMW E46 DME 诊断报文)              │
│  │                                               │
│  ├── 解码器 (DBC/规则)                           │
│  │   CAN ID → 物理值 (RPM=6500, TEMP=92°C...)   │
│  │                                               │
│  ├── 环形缓冲区 (最近 60 秒)                      │
│  │                                               │
│  └── DBus 广播                                    │
│      UI 订阅 → 仪表实时更新                       │
│      track-logger 订阅 → 赛道记录                 │
│      voice-engine 订阅 → "水温 92 度" 播报        │
└─────────────────────────────────────────────────┘
```

#### 4.2 传感器采集器

```
┌─────────────────────────────────────────────────┐
│           sensor-collector.service                │
│                                                  │
│  GPS 线程:  /dev/ttyS1 (UART)                  │
│  ├── 解析 NMEA 0183 (ATGM336H 10Hz Phase 1-5)   │
│  ├── Phase 6 升级 NEO-M9N 25Hz (改波特率即可)    │
│  ├── 提取: 经纬度、速度、高度、航向、时间          │
│  └── 10Hz → 25Hz 采样 → 共享内存环形缓冲         │
│                                                  │
│  IMU 线程:  /dev/spidev1.0 (SPI, 200Hz)         │
│  ├── ICM-42688 加速度 + 陀螺仪                   │
│  ├── 姿态解算 (Madgwick/Mahony，CPU)              │
│  └── 200Hz 采样 → 共享内存环形缓冲                │
│                                                  │
│  TPMS 线程: CC1101 (SPI) 或 BLE scan            │
│  ├── 315MHz: OOK 解码                            │
│  ├── BLE: BlueZ 被动扫描                          │
│  └── 周期性数据 (每 30-60s) → 共享内存            │
│                                                  │
│  电压监控: /sys/class/hwmon (ADC)               │
│  └── 1Hz 采样 → 过低告警                          │
│                                                  │
│  所有数据通过 DBus 广播                           │
└─────────────────────────────────────────────────┘
```

#### 5.2 电源管理

```
┌─────────────────────────────────────────────────┐
│              power-manager.service                │
│                                                  │
│  GPIO 监测 ACC/KL15 电平                          │
│                                                  │
│  ┌─────────────────────────────────────────┐     │
│  │ KL15 ON (开车)                           │     │
│  │ ├── 所有服务正常运行                      │     │
│  │ ├── CPU governor: ondemand               │     │
│  │ └── 温度监控: 每 5s 检查 /sys/thermal    │     │
│  │     >85°C → 降频到 1.2GHz               │     │
│  │     >95°C → 弹出高温警告 UI               │     │
│  └─────────────────────────────────────────┘     │
│                                                  │
│  ┌─────────────────────────────────────────┐     │
│  │ KL15 OFF → Suspend-to-idle               │     │
│  │ ├── 冻结所有用户进程                      │     │
│  │ ├── 保留 CAN 唤醒源 (PT-CAN 有活动)       │     │
│  │ ├── 保留 RTC 闹钟                         │     │
│  │ ├── 功耗: ~0.3W (50mA@5V)               │     │
│  │ └── 恢复延迟: <1s                        │     │
│  └─────────────────────────────────────────┘     │
│                                                  │
│  ┌─────────────────────────────────────────┐     │
│  │ Suspend > 16 分钟 → 深度关机              │     │
│  │ ├── 保存系统状态 (如果需要)                │     │
│  │ ├── 关闭所有外设供电 (GPIO 控制)           │     │
│  │ ├── 功耗: <0.01W                         │     │
│  │ └── 下次 KL15 ON → 冷启动 (5-8s)          │     │
│  └─────────────────────────────────────────┘     │
│                                                  │
│  电池保护:                                        │
│  ├── 常电电压 < 11.5V → 禁止进入 Suspend        │
│  ├── 常电电压 < 11.0V → 强制关机                 │
│  └── 防止 E46 电池亏电                           │
└─────────────────────────────────────────────────┘
